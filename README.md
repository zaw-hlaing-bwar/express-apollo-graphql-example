# Express apollo-graphql Example

This project is an example of usage for apollo-server-express in an express app.

## Installation

Install the dependencies with `npm` or `yarn`.

```bash
npm install
yarn
```

## Usage

Edit `index.js`.

Try adding new `Query` `Mutation`
